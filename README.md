# 🦋 Peppered Moth Evolution Simulator

A zero-install, browser-based evolution simulator for teaching natural selection. Watch moths adapt to pollution changes in real-time.

## ✨ Features

- **Zero install**: Just download `index.html` and open in any browser
- **Offline capable**: Runs entirely in the browser, no internet needed after download
- **Interactive**: Adjust pollution, predation, and mutation rates in real-time
- **Historical accuracy**: Models the famous UK peppered moth industrial melanism case
- **Three environments**: Clean forest, polluted industrial, or mixed habitat
- **Live charts**: Watch population dynamics unfold

## 🚀 Quick Start

1. Download `index.html` to any computer
2. Open in Chrome, Firefox, Safari, or Edge
3. Click "Start Simulation"
4. Watch evolution happen!

No accounts, no installs, no tracking.

## 📊 What It Teaches

- **Natural Selection**: Moths that contrast with tree bark get eaten
- **Industrial Melanism**: How UK pollution changed moth populations (1850-1950)
- **Adaptation**: How quickly populations can shift (~50-100 generations)
- **Habitat Diversity**: Mixed forests support both light and dark moths

## 🛠️ Technical Details

- Pure HTML/CSS/JavaScript - no frameworks
- Single file - easy to share and distribute
- Responsive design - works on Chromebooks, tablets, desktops
- Canvas-based rendering with real-time population charts

## 📖 The Science

The peppered moth (*Biston betularia*) is a classic example of natural selection:

- **1800s**: Light moths dominated (95%) - they camouflaged on clean trees
- **1850-1950**: Dark moths rose to 90% as industrial pollution darkened trees
- **1970s+**: Clean Air Acts restored light moth populations to 90%

This simulator lets students watch this process unfold in minutes instead of centuries.
